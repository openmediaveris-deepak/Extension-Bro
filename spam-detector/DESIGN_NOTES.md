# Spam Detector - Design System

## Color Palette

### Luxurious Black (Noir)
Inspired by Apple's dark mode, Uber's sophisticated UI, and Tesla's minimalist design.

- **noir-950**: `#000000` - Pure black for maximum contrast
- **noir-900**: `#0a0a0d` - Deep black for backgrounds
- **noir-800**: `#28282e` - Rich dark gray for cards
- **noir-700**: `#4d4d57` - Medium gray for borders
- **noir-400**: `#92929f` - Light gray for secondary text

### Accent Colors

**Gold** - Premium, luxury, success
- Primary: `#fbbf24`
- Dark: `#f59e0b`
- Usage: Highlights, important actions, success states

**Emerald** - Trust, security, active states
- Primary: `#10b981`
- Dark: `#059669`
- Usage: Status indicators, protection badges

## Typography

### Fonts
- **Primary**: Inter - Clean, modern, highly readable
- **Display**: SF Pro Display - Apple's signature font for headlines
- **Mono**: SF Mono - For code and technical content

### Hierarchy
- **H1**: 24px, Bold, Display font
- **Body**: 14px, Regular, Sans font
- **Caption**: 12px, Medium, Sans font

## Components

### Logo
- Shield shape symbolizing protection
- X mark for blocking spam
- Gold and emerald accents
- Subtle gradients and glows

### Layout
- Width: 400px
- Height: 600px
- Dark gradient background
- Subtle grid pattern overlay
- Ambient glow effects

### Effects
- Backdrop blur for glassmorphism
- Soft shadows with luxury variants
- Smooth transitions (200ms)
- Subtle hover states
- Glow effects on interactive elements

## Design Principles

1. **Minimalism** - Clean, uncluttered interface
2. **Sophistication** - Premium materials and effects
3. **Clarity** - Clear hierarchy and readable text
4. **Consistency** - Unified design language
5. **Accessibility** - High contrast, readable fonts

## Inspiration Sources

- **Apple**: Dark mode aesthetics, SF fonts, minimalist approach
- **Uber**: Sophisticated black UI, clean typography
- **Tesla**: Futuristic feel, high contrast
- **Stripe**: Premium gradients, subtle animations
